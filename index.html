Tecnologias: Power Bi, SQL MySQL, DAX, Excel

O que aprendi: Desenvolvimento de dashboards interativos no Power BI, criação de medidas com DAX, modelagem de dados, transformação de dados com SQL e aplicação de boas práticas de visualização e storytelling para análise orientada à decisão.
